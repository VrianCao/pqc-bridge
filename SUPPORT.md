# Support

PQC Bridge is in developer preview. The project is not ready for production
secrets until the roadmap reaches a reviewed backend and stable release process.

## Where To Ask

- General questions: GitHub Discussions, once enabled.
- Reproducible bugs: GitHub Issues.
- Feature proposals: GitHub Issues or Discussions.
- Security vulnerabilities: private vulnerability reporting or the process in
  [SECURITY.md](SECURITY.md).

## Response Expectations

Best-effort response targets for the preview phase:

- security reports: acknowledge within 72 hours
- reproducible bugs: initial triage within 7 days
- feature proposals: initial triage within 14 days

These targets are not a commercial support commitment.

## Unsupported Requests

The project does not provide:

- production deployment approval for v0.1
- legal advice
- FIPS 140-3 certification advice
- export-control advice
- custom cryptography design

## Before Opening an Issue

Please include:

- version or commit
- operating system
- language binding, if relevant
- exact command or code path
- expected behavior
- actual behavior
- minimal reproduction when possible
