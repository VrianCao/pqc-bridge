# Code of Conduct

PQC Bridge aims to be a serious, respectful, security-focused open source
project.

Participants are expected to:

- communicate respectfully
- focus criticism on code, design, and evidence
- avoid personal attacks
- disclose conflicts of interest when relevant
- handle security-sensitive information responsibly

Unacceptable behavior includes harassment, threats, discriminatory language,
deliberate disruption, and public disclosure of private vulnerability reports.

Maintainers may moderate issues, pull requests, discussions, and repository
access to protect the project and its contributors.
